# 版本更新

4.0以后的版本升级：发布新版本后，您只需要在您fork的仓库点击fetch即可更新到最新版本。（如果遇到冲突，尝试重新fork，或者手动解决）

![img.png](img.png)

4.0以前的版本升级：建议重新fork

如果使用docker或者server部署，更新后，后端需要重新部署。

4.2.0以后的版本，使用[@林木木](https://immmmm.com/hi-friends-circle/)的前端方案，在友链朋友圈的右下角可以检测更新情况~
